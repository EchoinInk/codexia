# Codexia Visual Identity

## Purpose

Defines the visual language for Codexia and Codier.


## Visual Direction

The brand should feel:

- futuristic
- intelligent
- premium
- approachable
- developer-focused


Avoid:

- generic SaaS aesthetics
- overly corporate visuals
- childish AI imagery


## Codier Design Principles

Codier should maintain:

- recognisable silhouette
- consistent proportions
- expressive but restrained emotions
- strong visual identity


## Expressions

Codier expressions should communicate:

- thinking
- building
- success
- curiosity
- guidance


## Colour Direction

Primary visual themes:

- deep dark backgrounds
- blue/violet energy
- soft highlights
- technology-inspired gradients


## Consistency

All Codier assets should feel like the same character across:

- poses
- illustrations
- UI states
- marketing assets
